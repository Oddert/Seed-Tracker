# Plotly Gantt Dash App

This workspace contains a simple Dash application that reads a `seed-tracker.csv` file
and displays a Gantt chart showing sowing, planting and harvesting windows.

## Setup

```powershell
python -m pip install -r requirements.txt    # or install dash, pandas, plotly manually
```

## Running

```powershell
python st-gantt-from-google-docs.py
```

Then open http://127.0.0.1:8050/ in your browser.
